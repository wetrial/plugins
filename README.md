## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.ts`,

``` ts
export default {
  plugins: [
    ['umi-plugin-module', options],
  ],
}
```
